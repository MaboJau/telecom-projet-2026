# telecom-projet-2026
Nice, 02/2026 — Master EEEA, ESTEL Dans ce projet, nous avons utilisé la carte ApolloEcho kit du LEAT afin de développer un module de renseignement sur la qualité de l’eau dans un environnement(plage, rivière, marigot). Il permet de fournir des données pour faciliter la prise de décision en contexte (sécurité des baigneurs, environnementale).
